## 一、系统简介

### 1.1 什么是资料通工程管理系统？

资料通工程管理系统是一款专为建筑施工企业打造的综合管理平台。系统集成了**物资管理**、**劳资管理**等多个核心业务板块，帮助企业实现工程项目的数字化管理，提升管理效率，降低运营风险。

### 1.2 核心优势

#### 🎯 一站式集成管理
将物资管理、劳资管理等多个业务板块集成在统一平台，告别多系统切换的烦恼。一个账号、一个界面，管理所有业务。

#### 🏢 项目部级数据隔离
支持合同→项目部→子项目三级组织结构，各项目部数据独立隔离，互不干扰。总部管理员可跨项目部查看全局数据。

#### 🤖 AI 智能助手
内置 AI 秘书功能，支持语音和文字查询。随时询问项目数据、物资库存、工资发放等信息，AI 自动搜索并返回结果。

#### 📱 多端同步使用
支持网站端、桌面端（Windows/Mac）、手机端（安卓/iOS/鸿蒙），随时随地管理工程项目。

#### 🔔 智能消息推送
接入钉钉、企业微信、飞书等主流办公平台，重要通知自动推送，定时发送项目报表给管理层。

#### 🛡️ 风险智能预警
劳资风控系统自动检测异常（如无考勤有发放、超龄用工等），红色/黄色分级预警，帮助企业规避用工风险。

#### 📊 数据可视化看板
直观的数据看板展示关键指标，支持多维度的数据分析和报表导出。

#### ☁️ SaaS 弹性订阅
按需订阅，灵活计费。支持单板块订阅和全系统会员，企业规模从小型到大型均可适配。

## 二、功能板块介绍

### 2.1 物资管理系统

> 管理工程项目的所有物资，从入库到出库全流程覆盖。

#### 核心功能

| 功能 | 说明 |
|------|------|
| 📥 入库管理 | 支持手动入库、OCR 智能识别送货单、Excel 批量导入 |
| 📤 出库管理 | 按项目和班组出库，自动生成 PDF 出库单 |
| 🔄 退库管理 | 支持从出库记录勾选退库，自动恢复库存 |
| 🔀 物资借调 | 跨项目借调物资，借调归还管理 |
| 📋 班组台账 | 各班组领料明细，自动计算净领料量 |
| ⚠️ 库存预警 | 低库存自动预警，避免物资短缺 |
| 📸 OCR 识别 | 拍照识别送货单，AI 自动解析物资信息 |

#### 使用场景

1. **物资入库**：供应商送货时，拍照送货单 → AI 自动识别 → 确认入库 → 自动更新库存
2. **物资出库**：选择项目 → 选择班组 → 选择物资 → 确认出库 → 自动生成出库单
3. **库存查询**：实时查看各项目物资库存，支持按物资名称、编码筛选
4. **班组对账**：查看各班组领料明细，导出 Excel 对账

---

### 2.2 劳资管理系统

> 管理工程项目的人员、考勤、工资和风控。

#### 核心功能

| 功能 | 说明 |
|------|------|
| 👥 人员管理 | 项目部人员、务工人员、分包商务工人员管理 |
| ⏰ 考勤管理 | 单条/批量录入考勤，支持加班记录 |
| 💰 工资核算 | 自动核算月薪/日薪，支持手动调整 |
| 💳 工资发放 | 手动录入或 AI 识别发放表，欠薪自动抵扣 |
| ⚠️ 风控管理 | 异常检测、合规预警、风控看板 |
| 📊 报表导出 | 多月合并 Excel 报表，4 个 Sheet 覆盖全维度 |
| 🤖 AI 识别 | AI 智能识别工资发放表（图片/PDF/Excel） |

#### 使用场景

1. **人员入职**：录入人员信息 → 身份证校验 → 合规检查（年龄、重复身份证）→ 自动预警
2. **考勤管理**：每日录入考勤 → 月度汇总 → 自动触发工资重算
3. **工资核算**：选择月份 → 自动核算 → 查看结果 → 手动调整（如需）→ 确认
4. **工资发放**：上传发放表 → AI 识别 → 确认记录 → 欠薪自动抵扣 → 异常检测
5. **风控监控**：查看异常列表 → 红色/黄色分级 → 取消异常（需填写原因）

### 2.3 合同与项目部管理

> 管理企业的合同和项目部组织结构。

#### 核心功能

| 功能 | 说明 |
|------|------|
| 📄 合同管理 | 承包合同和采购合同的统一管理 |
| 🏢 项目部管理 | 创建和管理项目部（分公司） |
| 📁 子项目管理 | 在项目部下创建和管理子项目 |
| 👥 成员管理 | 将用户分配到项目部，控制数据访问范围 |

#### 组织结构

```
合同（如：XX工程总承包合同）
  ├── 项目部一（如：XX项目部）
  │     ├── 子项目 A
  │     ├── 子项目 B
  │     └── 成员：张三（物资员）、李四（劳资员）
  └── 项目部二（如：YY项目部）
        ├── 子项目 C
        └── 成员：王五（物资员）、赵六（劳资员）
```
