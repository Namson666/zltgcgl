### 2026-05-19 — 阶段五：财务模块 Phase 1（基础数据 + 报账）
- Prisma Schema 新增 7 个 Fin 模型：FinCategory、FinSubCategory、FinPettyCashAccount、FinPettyCashAdvance、FinExpense、FinMonthlySummary、FinImportLog
- 新增 `backend/src/modules/finance/service.ts` — 财务模块业务逻辑层（~330 行）
  - 费用大类/子类 CRUD（软删除）
  - 费用记录 CRUD（分页、多条件筛选、关键词搜索）
  - 费用审核（通过/驳回）
  - 月度汇总快照刷新
  - 汇总/看板查询（月度趋势、分类汇总、总额统计）
  - 备用金账户查询
- 新增 `backend/src/modules/finance/routes.ts` — 财务模块 API 路由（~320 行）
  - `/api/finance/categories` — 费用大类管理
  - `/api/finance/sub-categories` — 费用子类管理
  - `/api/finance/expenses` — 费用记录管理（含审核）
  - `/api/finance/summary` — 汇总统计
  - `/api/finance/petty-cash-accounts` — 备用金账户查询
- `backend/src/app.ts` 注册 `/api/finance` 和 `/api/v1/finance` 路由
- 前端 API 新增 financeApi（12 个子模块，40+ 方法）
- 前端新建 4 个页面：ExpenseEntryDept（项目部报账）、ExpenseEntryFinance（公司财务凭证）、ExpenseList（费用列表）、CategorySettings（类别设置）

### 2026-05-19 — 阶段五：财务模块 Phase 3（汇总看板 + 导入导出）

### 2026-05-19 — 阶段五：财务模块 Phase 4（收款 + 开票）
