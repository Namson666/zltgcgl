# 资料通工程管理系统 - 开发者需求文档

> 本文档完整记录开发者关于系统的所有需求描述、功能规划和未来待开发需求。
> 最后更新时间：2026-04-24

---

## 一、系统概述

**系统名称**：资料通工程管理系统
**系统定位**：面向建筑施工企业的综合工程管理 SaaS 平台
**核心目标**：将物资管理、劳资管理等多个业务板块集成到统一平台，支持多端访问

---

## 二、核心需求

### 2.1 系统集成需求

- 将现有的物资管理系统（WMS）和劳资管理系统（Labor）合并为一个统一平台
- 统一数据库、统一认证、统一前端界面
- 模块化架构，方便后续添加新业务板块
- 共享基础数据（供应商、班组、员工等不重复建设）

### 2.2 多端支持需求

| 端 | 平台 | 技术方案 |
|----|------|---------|
| 网站端 | 浏览器 | React + Vite + TailwindCSS |
| 桌面端 | Windows / Mac | Tauri 2.x |
| 手机端 | 安卓 / iOS | React Native + Expo |
| 鸿蒙端 | HarmonyOS NEXT | ArkTS + ArkUI |

### 2.3 三方平台接入需求

- **钉钉**：扫码登录、工作台应用、消息推送
- **企业微信**：扫码登录、工作台应用、消息推送
- **飞书**：扫码登录、工作台应用、消息推送、Bot 机器人

### 2.4 AI 秘书需求

- 支持语音和文字输入
- 自然语言查询系统内数据（物资、考勤、工资等）
- 自动搜索数据库内的数据报表
- 通过微信（企业微信/普通微信）、飞书、钉钉发送报表
- 飞书可用 Bot 机器人沟通
- 定时向老板发送项目报表
- 普通微信通过类似 OpenClaw 的 Chatbot 实现

### 2.5 支付与计费需求

- 接入微信支付、支付宝线上支付
- 开发者自动开具发票功能
- 用户可自行注册（短信验证码）

### 2.6 SaaS 订阅计费需求

#### 订阅计划

| 计划类型 | 小型企业 | 中型企业 | 大型企业 |
|---------|---------|---------|---------|
| 单板块（物资/劳资） | 500元/月，5用户 | 10用户，每增1用户+50元/月 | 20用户，每增1用户+50元/月 |
| 全系统会员 | 888元/月，5用户 | 1288元/月，10用户 | 1888元/月，20用户 |
| 每增1用户 | +100元/月 | +100元/月 | +100元/月 |

#### 试用机制
- 注册后自动开启 30 天全功能试用期
- 到期前提醒续费
- 到期后保留数据但限制功能
- 按板块收费，未开通板块不显示

### 2.7 项目合同管理结构需求

```
合同（第一层）
  └── 项目部（第二层）── 数据隔离边界
        └── 项目名称/子项目
```

- 单个合同可设置多个项目部（分公司）
- 单个项目部可同时进行多个项目名称或子项目
- 项目部之间信息不互通
- 总部管理员、财务可对所有项目部进行权限选择

### 2.8 角色权限需求

| 角色 | 说明 | 默认数据范围 |
|------|------|-------------|
| 超级管理员 | 平台开发者 | 全平台 |
| 老板 | 企业负责人 | 全企业 |
| 管理员 | 企业管理员 | 全企业 |
| 物资员 | 物资管理 | 授权项目部 |
| 劳资员 | 劳资管理 | 授权项目部 |
| 财务 | 财务管理 | 全企业 |
| 出纳 | 出纳 | 全企业 |
| 项目出纳 | 项目出纳 | 所属项目部 |

- 默认权限在开发者后台设置，同步到所有用户
- 超级管理员可自行更改用户权限
- 特殊情况下可对劳资员、物资员开放更高层级权限（如总部级）

### 2.9 开发者后台需求

- AI 模型配置（大模型、识别类模型）
- 手动输入配置，点测试连通后自动启用
- 统一标注模型功能

### 2.10 长期记忆文档需求

## 三、现有系统功能清单

### 3.1 物资管理系统功能

- 入库管理：手动入库、OCR 智能入库、Excel 批量入库、送货单入库
- 出库管理：出库操作、PDF 出库单生成、出库撤销
- 退库管理：退库操作、Excel 批量退库
- 物资借调：跨项目借调、借调归还
- 班组台账：各班组领料明细
- 库存预警：低库存预警
- 操作日志：全模块操作记录
- 开发者后台：多租户管理、OCR/AI 配置

### 3.2 劳资管理系统功能

- 合同管理：承包合同 CRUD、进度款收款
- 分包商管理：分包商 CRUD
- 分包合同管理：分包合同 CRUD
- 产值管理：月度产值录入、分包进度款支付
- 人员管理：项目部人员、务工人员、分包商务工人员
- 考勤管理：单条/批量录入、加班记录、月度汇总
- 工资核算：自动核算（月薪/日薪）、手动修改
- 工资发放：手动录入、AI 智能识别、欠薪自动抵扣
- 风控管理：异常列表、合规预警、风控看板
- 报表导出：多月合并 Excel
